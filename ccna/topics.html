CCNA Exam Topics

Network Fundamentals

  • IP addressing and subnetting.
  • Basic network components (routers, switches, wireless, and endpoints).
  • OSI and TCP/IP models.

Network Access

  • VLANs, trunking, and inter-VLAN routing.
  • Spanning Tree Protocol (STP).
  • Wireless networking basics.

IP Connectivity

  • Routing protocols (static, RIP, OSPF, EIGRP).
  • IPv4 and IPv6 configurations.
  • Layer 3 redundancy (e.g., HSRP, VRRP).

IP Services

  • Network Address Translation (NAT).
  • Quality of Service (QoS).
  • DNS and DHCP.

Security Fundamentals

  • Basic network security concepts.
  • Firewalls and access control lists (ACLs).
  • Wireless security.

Automation and Programmability

  • Basics of network automation and orchestration.
  • Introduction to APIs and network programmability.
